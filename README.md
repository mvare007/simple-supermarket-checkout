
# Simple Supermarket Checkout


A simple cashier function that adds products to a cart and displays the total price.

## Run Locally

This project uses Ruby version `3.3.1`.
Assuming you are using rbenv:

Update rbenv and ruby-build, if you haven't:
```
  cd ~/.rbenv && git pull
  cd ~/.rbenv/plugins/ruby-build && git pull
```

Install `v3.3.1` and set it in your current working directory:
```bash
  rbenv install 3.3.1
  rbenv local 3.3.1
```
